Primeras Experiencias en Desafio Latam

En las primeras clases que he tomado desde que inicio el Desafio latam, hemos estado reforsando las bases de la programacion Web, desde el inicio comansamos a ver Comandos de Consola en Unix, Git, GitHub y HTML.

Comando con Unix

En las clases de Comandos con Unix nos dieron un enfoque a los comandos mas utilizados en consola, nos mostraron lo nesesario para poder administrar los archivos, visualizar, borrar, crear y explorar carpetas, crear Archivos. Posteriormente comentare mas a detalle lo que vimso en las clases.

Git

Posteriormente se vio Git, este es un Software de control de versiones. es muy util para tener un buen control en los cambios de tus proyectos, principalmente es muy util si se trabajan proyectos en equipos, Git te permite poder tener el historial de cambios en los codigos, regresarte a alguna version anteror para poder detectar errores o regresarte a alguna version donde si funcionaba bien el proyecto. En general se recomienda ampliamente el uso de Git en los desarrollos de Proyectos. En clase vimos los comandos basico, posteriorme hablare mas de lo que vimos sobre Git

GitHub

Si ya tienes Git, GitHub es una pagina Web que tienes que usar, GitHub es una plataforma de desarrollo colaborativo, este aloja proyectos que con los que estas usando Git. con GitHub puedes compartir tus repositorios, los haces publicos y tambien puedes bajar proyectos de otras personas.
